## Satori Highlights

Interactive architecture visualizer that turns your Flutter code into explorable diagrams:

- 🏗️ **Automatic Layer Classification** – View, State, Service, Model, Utility — detected automatically
- 🌐 **Three Visualization Modes** – Overview (layers), Focus (single class deep-dive), and List (search & filter)
- 🔗 **Relationship Analysis** – Inheritance, calls, dependencies, and data flow mapped visually
- 📦 **Package Intelligence** – External dependency analysis with automatic classification (SDK, official, third-party)
- 🎯 **Click-to-Code Navigation** – Jump from any diagram node straight to the source file
- 🎚️ **Abstraction Slider** – Explore from specific classes → folders → features → full project
